# Spade checker
Fast discord token checker

## tutorial
<=====================================================>
| Place your tokens in input/tokens.txt               |
| Each line = new token                               |
|  Example:                                           |
|          token examp                                |
|          token examp                                |
|          token examp                                |
| Etc                                                 |
| Run start.bat                                       |
| Put a delay                                         |
| Enjoy :3                                            |
<=====================================================>
